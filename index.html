SQL Between

Beteeen 接受兩個值之間的判斷(數字、文字、日期)

用數字來判斷
SELECT * FROM books WHERE year BETWEEN 2000 AND 2020
此結果會包含2000-2020的年份

用字串來判斷，注意大小寫有區別
SELECT * FROM books WHERE name BETWEEN 'Ａ' AND 'Ｄ'
會包含Ａ、B、C 開頭的書名，不會包含Ｄ開頭

也可以用AND加入更多的判斷
SELECT * FROM books WHERE year BETWEEN 2000 AND 2020 AND neme = 'SQL'

Magento order invoice prefix

Magento Version 2.2

如果我們想要在訂單前面加上前綴
e.g. SOR0000001

一. 首先到 DB sales_sequence_meta 確認 order 的 meta_id

二. 確認後，再到 sales_sequence_profile 加入前綴

e.g.

UPDATE "sales_sequence_profile" SET "prefix" = "SOR" WHERE "meta_id" = 5;
//這裡的 `meta_id 5` 對應 `sales_sequence_meta` 是 order

  • SOR0000001

PHP print()、print_r()、echo() 差別

PHP Version 7

echo

  • 無返回型態(type)

      echo $value,$value2;

print

  • 有返回型態(type) always return 1

  • 可以用來做複雜的表達式

      print $value.value2;
  $x ? print "true" : print "false";

print_r

  • 用來輸出陣列

      print_r ( $array , bool $return )//$return = true 則不輸出資訊,預設為false

  $array = array('test' => '123', 'test2' => '456');

  print_r($array);

  Array
  (
      [test] => 123
      [test2] => 456
  )
